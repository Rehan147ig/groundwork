# docs/business

Business and go-to-market assets for Groundwork. These are deck-grade artifacts, not technical specs — pitch decks, decision matrices, market positioning material.

## Index

| File | What it is | When to use it |
| --- | --- | --- |
| `bank-stack-decision-matrix.html` | Phase A decision matrix comparing Apache Fineract, Temenos, Mambu, Open Bank Project, and the synthetic Groundwork baseline. Explains the Push vs Pull vs Hybrid integration architecture. | Embed or screenshot into a deck. Send to a bank prospect debating which integration path applies to them. |

## Editing rules

- Anything in this directory is intended to be **shown to an external audience** — investors, bank security buyers, journalists. Treat every claim as if a regulator may verify it.
- Cross-reference numerical claims against the canonical knowledgebase document (`Groundwork — Pitch & Validation Knowledgebase`). Don't invent stats.
- HTML files are self-contained (Google Fonts via CDN only). No external CSS or build step.
