# Examples

This folder is for demo clients and integration proofs that exercise Groundwork without becoming part of the core runtime.

Current examples:

- `hyperagent-groundwork-demo.mjs` — HyperAgent-oriented MCP demo helper.
- `mcp/groundwork-mcp.js` — older lightweight JavaScript MCP wrapper retained as an example.

Planned examples:

- `bank-demo/` — synthetic banking documents, personas, OpenFGA permissions, seed scripts, and Java Spring Boot client.
